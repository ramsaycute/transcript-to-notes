# 📝 Transcript to Notes

一个 Agent Skill，将录音转录文本（会议、课程、问诊、访谈等）一站式整理为结构化笔记。

## ✨ 功能

- **自动预处理**：检测并修复转录文本的格式问题（单行过长、标点空格异常等）
- **语义分段**：按主题自动划分段落，提炼小标题
- **同音字纠错**：结合语境和领域知识，修正语音识别中的同音字 / 近音字错误
- **口语转书面**：将口语化表达整理为清晰的书面语，保留所有有实质意义的细节
- **长文本支持**：分批处理策略，稳定处理长篇转录文本

## 📁 项目结构

```
transcript-to-notes/
├── SKILL.md            # Skill 定义文件（工作流程与指令）
└── scripts/
    └── preprocess.py   # 文本预处理脚本
```

## 🚀 使用方式

这是一个 Agent Skill，适用于支持 Skill 机制的编码代理（如 Claude Code、OpenCode 等）。安装后，当你提供录音转录文本并要求整理为笔记时，会自动触发。

### 安装

将本项目克隆到你的 skills 目录：

```bash
git clone <repo-url> ~/.claude/skills/transcript-to-notes
```

### 触发方式

打开一个转录文本的 Markdown 文件，然后要求 AI 将其整理为笔记即可。

### 工作流程

1. **预处理阶段**：运行 `scripts/preprocess.py` 对原始转录文本进行格式清理，将清理后的文件输出到 `_cleaned` 目录
2. **语义整理阶段**：
   - 读取开头，识别场景、领域和说话人
   - 分批读取、整理并写入：语义分段 → 同音字纠错 → 口语转书面
   - 将结构化笔记输出到 `_notes` 目录，文件名为 `*_notes.md`

## 📋 环境要求

- Python 3
- 支持 Skill 的编码代理

## 📄 输出示例

整理后的笔记格式如下：

```markdown
# 主题标题

**场景**：课程 / 会议 / 问诊 / 访谈等
**领域**：所涉及的领域
**参与者**：人数及姓名 / 角色
**概要**：3-5 句话概括核心内容

---

## 段落小标题

整理后的段落内容...
```
